## NSS driver feature notes

- CAPWAP: 未测试是否支持，CAPWAP 数据隧道的 DTLS 加解密。
- C2C: 未测试是否支持，两颗 IPQ8074 通过 PCIe 互联，NSS 引擎协同做包处理。
- CLMAP: 未测试是否支持，企业 WiFi 场景下 MAC 地址到隧道的映射表。
- DTLS: 未测试是否支持。
- IPSEC: 未测试是否支持。
- OAM: 电信级设备的管理协议。
- PORTID: 交换机端口映射。
- QVPN: 未补充说明。
- UDP_ST: 测速工具，测试 NSS 纯硬件转发极限。
- TLS: 硬件 TLS 卸载。
- TSTAMP: 硬件时间戳。
- WIFI_LEGACY: ipq806x、ipq40xx 用于老 WiFi 接口。
